# Prompt dùng trong Codex

Sau khi copy hoặc giải nén thư mục này trên máy Windows, mở Codex tại đúng thư mục đó và gửi prompt:

```text
Hãy cài và chạy bộ Zalo PC CSV Sender trong thư mục hiện tại.

Yêu cầu:
1. Chạy .\install_windows.ps1 để cài Python và dependency nếu thiếu.
2. Kiểm tra contacts.csv bằng .\run_check.ps1.
3. Nhắc tôi mở Zalo PC và đăng nhập.
4. Nếu tôi yêu cầu chạy có xác nhận thì chạy .\run_confirm.ps1.
5. Nếu tôi yêu cầu chạy tự động chậm thì chạy .\run_auto_slow.ps1.
6. Nếu có lỗi, đọc zalo_sender.log và sửa script.
```
